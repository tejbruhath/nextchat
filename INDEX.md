## üìã Quick Reference

### URLs
- Frontend: `http://localhost:5173`
- Backend: `http://localhost:5000`
- Health Check: `http://localhost:5000/api/health`

### File Locations
- Backend: `./backend/`
- Frontend: `./frontend/`
- Database: `./backend/chat.db`
- Uploads: `./backend/uploads/`
- Config: `./backend/.env`

### Key Commands
```bash
# Start everything
npm run dev

# Backend only
cd backend && npm start

# Frontend only
cd frontend && npm run dev

# Install all
npm run install-all
```
