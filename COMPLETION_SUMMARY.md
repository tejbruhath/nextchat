## 🚀 How to Start

### Option 1: Windows Quick Start
```bash
# Double-click this file:
start.bat
```

### Option 2: Manual Start
```bash
# Terminal 1 - Backend
cd backend
npm install
npm start

# Terminal 2 - Frontend
cd frontend
npm install
npm run dev
```

### Option 3: All at Once
```bash
# From root directory
npm run install-all
npm run dev
```

## 🐛 Troubleshooting

### Common Issues

2. **Dependencies**: Run `npm install`
3. **Database**: Delete `chat.db` to reset
4. **Connection**: Check both servers running
